06 — questions

Good things to know.

How does it only show on the right page?

Otto reads which service the code is for — from the sender and the wording of the email (“your code to sign in to SuperDM”). It then offers the code only on a page that matches. Switch tabs and the chip stays behind where it belongs.

Does my Gmail have to be open?

It needs to be open as a background tab — you never have to look at it. You stay on the login page while Otto watches your inbox quietly behind it.

Is my email safe? Where does the code go?

Nowhere but your own browser. Otto reads the code from your open Gmail tab and keeps it locally for a few minutes. No servers, nothing uploaded — and it's open source, so you can verify exactly that.

What if the same code is sent again?

Once you use a code, Otto retires it. If the site re-sends — even with identical digits — Otto stamps the new arrival and treats it as a fresh, active code you can use again.

Which browsers work?

Any Chromium-based browser: Chrome, Arc, Edge, and Brave. If it runs Chrome extensions, it runs Otto.

What about Outlook, iCloud or Proton?

Gmail is supported today. Other providers are on the roadmap — the architecture makes each one a small add-on. Want one next? Open an issue on GitHub.
